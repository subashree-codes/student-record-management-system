# Student Record Management System

A command-line based Student Record Management System developed using Python.  
This project allows users to manage student records efficiently through a menu-driven interface.

## Features

- Add student records
- View all students
- Search students by name or USN
- Delete student records
- Automatic grade calculation based on marks
- Persistent data storage using file handling

## Technologies Used

- Python
- Git & GitHub

## Concepts Implemented

- Functions
- Loops and Conditional Statements
- Lists and Dictionaries
- CRUD Operations
- File Handling
- Data Persistence

## How to Run

1. Open the project folder in VS Code
2. Open terminal in the project directory
3. Run the following command:

python main.py
